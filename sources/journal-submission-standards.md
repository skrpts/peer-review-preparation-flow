## Journal Submission Standards Reference

### Universal Submission Requirements

Most peer-reviewed journals require the following, regardless of discipline:

#### Manuscript Components
| Component | Typical Requirement |
|-----------|-------------------|
| Title page | Title, all authors with affiliations, corresponding author contact, word count, running head |
| Abstract | 150-300 words, structured (Background, Methods, Results, Conclusion) or unstructured |
| Keywords | 4-6 keywords not already in the title |
| Main text | Introduction, Methods, Results, Discussion (IMRaD) for empirical work |
| References | Formatted in the journal's required style |
| Tables | Each on a separate page, numbered sequentially, with descriptive captions |
| Figures | High resolution (300+ DPI), acceptable formats (TIFF, EPS, PDF, high-quality JPEG) |
| Supplementary materials | Clearly labelled and referenced in the main text |

#### Ethical and Compliance Statements
| Statement | When Required |
|-----------|--------------|
| Ethics approval | All research involving human participants, animals, or identifiable data |
| Informed consent | Studies involving human participants |
| Conflict of interest disclosure | Always (declare conflicts or state "none") |
| Funding statement | Always (list funders and grant numbers or state "no funding") |
| Data availability statement | Increasingly required — state where data can be accessed |
| Author contributions (CRediT) | Increasingly required — use CRediT taxonomy |
| AI/LLM disclosure | Increasingly required — describe how AI tools were used |

### Discipline-Specific Standards

#### Social Sciences and Psychology
- **Reporting standard:** APA 7th edition style
- **Word count:** 5,000-8,000 words (excluding references)
- **Statistical reporting:** APA format — F(df1, df2) = value, p = value, effect size
- **Common reporting guidelines:** CONSORT (RCTs), STROBE (observational), COREQ (qualitative)
- **Pre-registration:** Increasingly expected for confirmatory studies
- **Sample size justification:** Power analysis expected

#### Health Sciences and Medicine
- **Reporting standard:** Vancouver style (ICMJE recommendations)
- **Word count:** 3,000-5,000 words for original research
- **Common reporting guidelines:** CONSORT, PRISMA (systematic reviews), STARD (diagnostic accuracy), MOOSE (meta-analyses)
- **Trial registration:** Required before enrolment for clinical trials
- **Ethics:** Full IRB/ethics committee approval details required
- **ORCID:** Often mandatory for all authors

#### Humanities and Arts
- **Reporting standard:** Chicago or MLA style
- **Word count:** 6,000-12,000 words (often longer than sciences)
- **Structure:** More flexible — not required to follow IMRaD
- **References:** Often extensive — 50-100+ references expected
- **Figures:** Less common; when used, may have specific copyright considerations

#### Engineering and Computer Science
- **Reporting standard:** IEEE style
- **Word count:** 4,000-8,000 words (conference papers: 6-10 pages)
- **Code availability:** Increasingly expected — GitHub or institutional repository
- **Reproducibility:** Methods must enable replication; sharing of code and datasets encouraged
- **Conference vs. journal:** Conference publications carry significant weight in CS

### Common Reporting Guidelines

| Guideline | Study Type | Key Requirements |
|-----------|-----------|-----------------|
| CONSORT | Randomised controlled trials | Flow diagram, participant numbers at each stage, ITT analysis |
| STROBE | Observational studies | Study design in title, eligibility criteria, sources of bias |
| PRISMA | Systematic reviews | Search strategy, PRISMA flow diagram, risk of bias assessment |
| COREQ | Qualitative research | Researcher characteristics, relationship with participants, analysis method |
| ARRIVE | Animal research | Species, strain, sex, housing, sample size justification |
| SRQR | Qualitative research | Reflexivity, context, data collection, analysis approach |
| TRIPOD | Prediction models | Development vs. validation, model specification, performance measures |
| CARE | Case reports | Timeline, diagnostic assessment, interventions, outcomes |

### Manuscript Length Guidelines

| Journal Tier | Article Type | Typical Word Limit | Reference Limit |
|-------------|-------------|-------------------|----------------|
| High impact (Nature, Science, Lancet) | Article | 3,000-5,000 | 30-50 |
| Mid-tier discipline journals | Original research | 5,000-8,000 | 40-60 |
| Specialist journals | Original research | 6,000-10,000 | 50-80 |
| Review journals | Review article | 8,000-15,000 | 80-150 |
| Any | Letter/Brief report | 1,500-3,000 | 15-25 |
| Any | Commentary/Editorial | 1,000-2,000 | 10-15 |

### Figure and Table Standards

| Requirement | Standard |
|-------------|---------|
| Resolution | 300 DPI minimum for photographs, 600 DPI for line art |
| Colour model | CMYK for print, RGB acceptable for online-only |
| File formats | TIFF, EPS, or PDF preferred; high-quality JPEG acceptable |
| Maximum file size | Typically 10-20 MB per figure |
| Font size in figures | Minimum 8pt after scaling to final reproduction size |
| Accessibility | Must be interpretable in greyscale; colourblind-friendly palettes recommended |
| Permissions | Written permission required for any reproduced figures |
